ETIQUETAS DE LA BASE

A continuacion vamos a tener las etiquetas para el uso de la base en la creacion de una pagina Web

Etiqueta Observacion
<!doctype> Esta etiqueta Nos indica que version de Html Estamos utilizando
<html> Esta etiqueta es la Raiz del Archivo donde se inicia el codigo para tu pagina web.
<Title> Esta estiqueta nos permite darle un Nombre a la Pagian.
<Head> El elemento de HTML Header "header" representa un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezado, así como también un logo, un formulario de búsqueda, un nombre de autor y otros componentes.
<Link> Esta etiqueta nos sirve para enlazar los archivos con los de CSS y poder asi aplicar los estilos.
<Style> Esta etiqueta nor permite dar estilo CSS direntamente a la etiquetas.
<Script> Esta etiqueta nospermite enlazar y escribir codigo JavaScrip dentro de Html.
<meta> Esta etiqueta se utiliza para porder dar le caracteristicas a la pagina.
<body> Esta tiqueta nos representa el cuerpo de la pagina, a demas dentro de esta es donde podemos utilizar las demas.

>

ETIQUETAS PARA EL TEXTO

Las siguientes etiquetas son las que se utiliza para poder plasmar texto en el body, a demas podemos dar le formato, a todas las paginas en html que quieras diseñar para todos tus proyectos.

Etiqueta Observacion
<p> Esta etiqueta la utilizamos para crear parrafos en nues tra paginas.
<h1>,<h2>,<h3>,<h4>,<h5>,<h6> Esta etiqueta se utiliza en el momento en el que quieras dar el titulos y subtitulos el cual se tiene una jerarquia en pesando con la <h1>.
<b> Esta estiqueta nos permite aplicar al texto el formato negrillas, sin utilizar el Css.
<i> Esta etiqueta nos permite dar le formato de Cursiva a nuestro texto.
<u> Esta etiqueta nos permite dar le formato de Subrayado a nuestro texto.
<a> Esta etiqueta nos permite agregar un link a nuestra textos o imagenes.
<strong> Con esta etiqueta podemos indicar que el texto es importante en la pagina.
<blockquote> Esta etiqueta nos permite dar como formato de cita al texto que queramos.
<br> Esta tiqueta nos permite crear dentro de la etiqueta <body> saltos de lineas, bueno donde la quieras utilizar.
>

ETIQUETAS PARA UTILIZAR MULTIMEDIA

Las siguientes etiquetas son las que se utiliza para poder plasmar texto en el body, a demas podemos dar le formato, a todas las paginas en html que quieras diseñar para todos tus proyectos.

Etiqueta Observacion

<video> Esta etiqueta se utiliza como su nombre lo indica solo para insertar algun video en nuestra pagina.

ETIQUETAS PARA ESTRUCTURAR EL CONTENIDO

Las siguientes etiquetas podemos darle al texto las imagenes estamos hablando de casi todo lo que coloques en tu paginas web, una las funciones de estas etiquetas es poder centrar tu contenido como la etiqueta <div> que nos permite esto, tan bien desde crear listas ordenadas entre mas cosas.

Etiqueta Observacion
<ul> Esta etiqueta la utilizamos para crear listas, pero en el momento no maneja ningun ordenamiento.
<ol> Esta etiqueta se utiliza para crear lista, a diferencia la etiqueta <ul> con esta si podemos ordenar el contenido.
<li> Esta estiqueta nos permite difinir articulos de las listas y poder trabajar con ellos.
<div> Esta etiqueta nos permite agrupar contenido o seccionar todas las etriquetas estas contenga a si se nos sera mas facil aplicar y formatear las estructura.
<nav> Esta etiqueta nos permite crear menus para navergar por las diferentes paginas que tengamos en nuestro proyectos.
<header> Esta etiqueta podemos indicar cual va hacer nuetra cabeceras en la pagina.
<footer> Con esta etiqueta podemos indicar cual va hacer el pie de la paguina, para ser mas claro lo ultimo que nuestra pagina va tener para mostrar.
>

ETIQUETAS PARA LA CREACION DE TABLAS

Las siguientes etiquetas podemos utilizar la para creacion de las tablas, tan bien podemos mostrar informacion al usuario ya sea estatica o consultada de una base de datos, a nos sirve tan bien para para darle formato a nuestras tablas.

Etiqueta Observacion
<table> Esta etiqueta la utilizamos para crear tablas en nuertra pagina.
<tr> Esta etiqueta se utiliza para crear las filas te va tener nuestra tabla.
<td> Esta estiqueta nos permite digitar la informacion que tendar los campos de la tabla a mostrar, tanto como agragar nuevas columnas a nuetra tabla.
<th> Esta etiqueta nos permite agrupar el titulo de las tablas.
<col> Esta etiqueta nos permite especificar las propiedades que va a tener las columnas.
>

ETIQUETAS PARA LA CREACION DE FORMULARIOS

Las siguientes etiquetas se utlizan para crear formularios, como para restros o algunas secciones de para los usuarios que visiten tu pagina.

Etiqueta Observacion
<form> Esta etiqueta la utilizamos para crear formularios.
<input> Esta etiqueta se utiliza para crear campos de datos, donde podran diligenciar la informacion o mostar.

<textarea> Esta etiqueta nos permite crear cajas para ingresar texto grandes, como observaciones parafos largos, entre otros.
<button> Esta etiqueta nos permite crear los botones para el formulario.
